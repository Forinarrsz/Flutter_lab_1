Данное приложение представляет собой менеджер задач (ToDo list), работающий через консольный интерфейс. Проект демонстрирует основные возможности Dart: Null Safety, асинхронность, работу с коллекциями и внешними пакетами.

## Ответы на контрольные вопросы

   1. Чем final отличается от const?
   final это значение устанавливается один раз и известно только во время выполнения (runtime).
   const — константа времени компиляции (compile-time), значение должно быть известно заранее.
   2. Что означает String?
   Это тип данных, который может содержать либо строку, либо значение null (nullable тип).
   3. Чем Future отличается от обычного значения? Что означает await?
   Future  это «обещание» результата, который будет получен позже. await приостанавливает выполнение функции до завершения Future, не блокируя при этом основной поток выполнения.
   4. Зачем в Dart именованные конструкторы?
   В Dart отсутствует классическая перегрузка методов и конструкторов, поэтому именованные конструкторы используются для создания объектов разными способами.
